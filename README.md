# 📦 SARAYA Monthly Lot Analysis Dashboard

**Real-Time Warehouse Management System**

## 📊 Dashboard Overview

Professional warehouse inventory dashboard for SARAYA MFG. (Thailand)

### Key Features
✅ **100% Real Data** - Extracted from Excel warehouse inventory  
✅ **Interactive Charts** - Chart.js powered visualizations  
✅ **Smart Search** - Find products by code or name  
✅ **Duration Analysis** - Storage time categorization  
✅ **Professional Design** - Modern gradient UI  
✅ **Tab Navigation** - 2605xx & 2606xx monthly lots
